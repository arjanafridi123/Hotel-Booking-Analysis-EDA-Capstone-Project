# Hotel-Booking-Analysis-EDA-Capstone-Project

**About The Project**

This project focuses on performing Exploratory Data Analysis (EDA) on hotel booking data collected from City Hotels and Resort Hotels. The dataset contains information related to booking status, arrival dates, customer types, market segments, distribution channels, deposit policies, country of origin, and Average Daily Rate (ADR).

The purpose of this analysis is to understand customer booking behavior, identify cancellation patterns, evaluate pricing trends, analyze market segment performance, and examine geographical demand. Through data cleaning, visualization, and statistical analysis, the project generates actionable business insights that can help hotel management improve occupancy, optimize pricing strategies, reduce cancellations, and enhance overall operational performance.
